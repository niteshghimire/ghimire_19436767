W3schools:

W3schools has been providing finest support to students who are new to coding. It’s easy to learn coding referring this site. This site provides wide range of IT knowledge for free of cost to anyone who browse it. Some of the courses this site provides are: HTML and CSS, Java Script, python, Web building etc. As a regular visitor of this site, I suggest you all to once visit it.

CSS Tricks:

CSS Tricks is another website to visit for learning codes and gaining knowledge about IT subjects. I have studied about flexbox in this website and it was an amazing experience using it, since its well explained in well simple manner. This site is also best for learning web developments.

Codecademy:

Codecademy is best online site for beginners. It provides platform to practice codes. This website provides some tabs when we go through it: An instruction tab, codes writing tab and execution tab. We can learn use of code instantly. First read instructions and then write codes and see the result in execution tab. Everyone should visit this site, it’s good.
