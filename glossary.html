HTML

“Full form of HTML is Hyper Text Markup Language. It is the standard markup language for creating Web pages. It describes the structure of Web pages using markup. HTML elements are the building blocks of HTML pages. HTML elements are represented by tags.” For more visit W3schools. (W3schools.com, 2019)

CSS

“Full form of CSS is Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once like “we can give styles for header and footer in single CSS page for many html pages”. External stylesheets are stored in CSS files and are linked in html files.” For more visit W3schools. (W3schools.com, 2019)

Java Script

“JavaScript is the Programming Language for the Web. It can update and change both HTML and CSS. JavaScript can calculate, manipulate and validate data.” For more visit W3schools. (W3schools.com, 2019)
