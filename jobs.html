Front end developer

They are responsible for the presentation and layout of website. They work with tools like photoshop and language like html CSS and JS. Frontend refers to "client-side". Main objective of frontend developer is to make user friendly design.

Backend developer

They are responsible for proper functioning of website They work with tools like XAMPP and language like php. Backend refers to "server-side". Main objective of backend developer is to make site fast and reliable.
