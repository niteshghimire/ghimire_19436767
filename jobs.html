FrontEnd Developer

  1. They are responsible for the presentaion and layout of website
  2. They work with tools like photoshop and language like html css and js
  3. Frontend refers to "client-side"
  4. Main objective of frontend developer is to make user friendly design

BackEnd Developer

  1. They are responsible for proper functioning of website
  2. They work with tools like xampp and languwage like php
  3. Backend refers to "server-side"
  4. Main objective of backend developer is to make site fast and reliable
